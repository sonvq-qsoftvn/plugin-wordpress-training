User Management Plugin that creates user registration, login, change password, forgot password form using ajax. Simple and customizable.
== Description ==
* The plugin allows users to create user forms via shortcode in the simpliest way.
* Features:
1. Create forms on front-end for user registration, login, change password, forgot password
2. Validate data
3. Easy to manage users on front-end
4. Easy to customize, install and maintain
5. Add custom field
6. Fully ajax
7. Facebook login
8. Not css
9. Manage email template

== Installation ==
1. Upload "Manage user ajax simple" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Create pages and insert shortcode(click button create auto)

* login: [qsoft_login_form] 
* reset password: [qsoft_reset_password_form] 
* Input new password: [qsoft_input_new_password_form] 
* Change password: [qsoft_change_password_form] 
* Register and manage user: [qsoft_registration_form]
